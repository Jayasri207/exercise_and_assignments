Exercise -5 Installation of Kubernetes Minikube Single node cluster
1)
  a) Download minikube binaries, follow below steps.

  $ sudo apt install -y curl wget apt-transport-https conntrack

  $ curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64

  b) install minikube binaries

   $ sudo install minikube-linux-amd64 /usr/local/bin/minikube

 # verify minikube version

   $ minikube version

  c) install kubectl utility

   $ curl -LO https://storage.googleapis.com/kubernetes-release/release/`curl -s https://storage.googleapis.com/kubernetes-release/release/stable.txt`/bin/linux/amd64/kubectl

   $ chmod +x kubectl

   $ sudo mv kubectl /usr/local/bin/

  d) start minikube

   $ minikube start --vm-driver=docker --force

   $ minikube status

2) Push Docker container images to Docker registry

a) #On the Docker Host at the command prompt run command to login to docker registry.

  $ sudo docker login
# provide username and password at prompt.

b) Tag the Image with registry identity and push image to docker registry

$ sudo docker tag webserver:latest ganeshhp/webserver:07182023

$ sudo docker push ganeshhp/webserver:07182023
